# DocPilot README 截图清单

建议统一使用 1600×900 或 1920×1080 浏览器窗口，浏览器缩放保持 100%。截图中不要出现真实 API Key、个人路径、终端历史或私人文档。

1. `01-workspace-overview.png`
   - 使用 `manager` 登录并打开含文档的知识库。
   - 展示文档数、切片数、问答数、当前模型、检索模式和向量索引覆盖率。

2. `02-chat-and-sources.png`
   - 左侧至少显示两个独立对话，并选中其中一个。
   - 提问一个能命中文档的问题，展开引用来源。
   - 画面中同时包含会话列表、历史消息、回答、原文件名、原文片段和相关度。

3. `03-document-pipeline.png`
   - 展示文档列表、解析状态和一份文档的切片详情。
   - 如果单张图放不下，可以将列表和详情拼成一张图。

4. `04-member-permissions.png`
   - 使用 `manager` 登录。
   - 展示负责人、仅使用成员、可管理成员和右侧授权表单。

5. `05-model-settings.png`
   - 使用 `admin` 登录。
   - 展示回答模式、Embedding 模式、模型名称、连接检测、索引覆盖率和重建按钮。
   - 不要截取任何真实云端 API Key。

6. `06-login.png`
   - 展示完整登录页，不要保留浏览器收藏栏或个人账号信息。

截图完成后，将图片放入本目录，并取消根目录 README“项目预览”中的 HTML 注释。
