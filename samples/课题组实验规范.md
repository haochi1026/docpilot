# 智能系统课题组实验规范（演示资料）

## 设备预约

GPU 工作站实行预约使用。成员应至少提前一天填写用途和预计时长，单次预约原则上不超过四小时。预约开始后十五分钟仍未签到，系统可以释放该时段供其他成员使用。

## 数据管理

实验原始数据保存在课题组对象存储中，不得仅保存在个人电脑。数据目录应包含 README，说明数据来源、采集日期、处理脚本版本和负责人。涉及个人信息的数据必须完成脱敏后再进入共享目录。

## 实验记录

每次模型训练需要记录代码提交号、数据集版本、随机种子、主要超参数和评价指标。实验失败也应记录原因，避免其他成员重复消耗算力。

## 论文与引用

内部论文草稿不得上传到公开网盘。引用实验结果时应链接到对应的实验记录，并由至少一名成员复核。对外发布前需要确认数据授权和第三方代码许可证。

## 安全要求

不得共享个人账号和访问令牌。发现密钥进入 Git 历史后，应立即撤销密钥，而不是只删除最新提交中的文本。离开实验室前应确认移动设备断电、门窗关闭。
